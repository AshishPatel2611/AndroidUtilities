# Usage
Gradle
Add these lines on top-level build file
```
repositories {
    maven {
        url "https://jitpack.io"
    }
}
```
On app's build.gradle
```
implementation 'com.github.AshishPatel2611:AndroidUtilities:0.0.1'
```
